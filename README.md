# Fairbnb

> *Airbnb clone*

#### 'Fairbnb' is a place for users to discover awesome bed and breakfast spots posted by others. Checkout the fair booking prices and plan your next stay. *Design inspired by Airbnb.* 

# Technologies Used
- React
- Redux
- Express.js
- PostgreSQL
- Sequelize
